Rock Paper Scissors Game

A Python-based implementation of the classic Rock Paper Scissors game. This project demonstrates fundamental programming concepts such as user input handling, conditional statements, and random number generation.

Features

  Interactive command-line gameplay
  Random computer choice generation
  Automatic winner determination
  Simple and user-friendly interface
